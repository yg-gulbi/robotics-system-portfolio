# Tech Stack

기술 스택은 단순 나열이 아니라 현재 수준, 사용 경험, 관련 프로젝트, 다음 보완점을 함께 정리합니다.

## 레벨 기준

| Level | 기준 |
| --- | --- |
| 1~2 | 개념을 들어봤거나 아주 기초만 아는 수준 |
| 3~4 | 기본 문법이나 사용법을 따라 할 수 있는 수준 |
| 5~6 | 작은 프로젝트나 실습에 적용할 수 있는 수준 |
| 7~8 | 문제를 해결하면서 독립적으로 활용할 수 있는 수준 |
| 9~10 | 실무 수준으로 설계, 최적화, 디버깅, 설명까지 가능한 수준 |

현재 레벨은 보수적으로 적고, 실제 프로젝트나 실습 근거가 생기면 갱신합니다.

# Study Notes

공부한 내용을 프로젝트처럼 포장하지 않고, 현재 이해한 범위와 다음에 검증할 내용을 정리하는 영역입니다. 실제 결과물이 있는 경우에는 대표 프로젝트로 연결하고, 그렇지 않은 내용은 학습 지도와 실습 계획으로 둡니다.

## 영역

- [computer-architecture-os](computer-architecture-os/README.md): 컴퓨터구조와 운영체제 개념을 C/Linux 실습으로 확인하기 위한 기록
- [control-theory](control-theory/README.md): PID, 상태공간, body feedback, speed loop를 균형 로봇 자세 제어와 연결
- [robotics](robotics/README.md): ROS, Gazebo, 센서, SLAM, Navigation 학습 기록
- [embedded-system](embedded-system/README.md): Arduino, 센서 입력, 모터 제어, 통신, 실시간 제어 기초

## 작성 기준

- 학습 목적
- 현재 이해한 범위
- 실습 또는 적용 근거
- 실제 프로젝트와의 연결점
- 다음 보완점

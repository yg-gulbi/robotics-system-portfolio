# Embedded System

## 학습 목적

Arduino, 센서 입력, 모터 제어, 통신, 실시간 제어 기초를 실제 로봇 제작 경험과 연결해 정리합니다.

## 핵심 개념

- Arduino 기반 입출력
- IMU 센서 데이터 수집
- BLDC 모터 제어
- ODrive 연동
- 통신 기초
- 실시간 제어 기초

## 정리 내용

임베디드 시스템은 하드웨어의 입력과 출력을 시간 조건 안에서 다루는 것이 중요합니다. 밸런스 로봇 프로젝트에서는 IMU 데이터를 읽고 제어 입력을 계산한 뒤 모터 구동으로 연결하는 흐름을 중심으로 정리합니다.

## 실습 계획

- IMU 데이터 읽기와 보정
- 센서 노이즈 관찰
- 모터 제어 입력 변화 실험
- Arduino와 ODrive 연동 정리
- 통신 지연과 제어 안정성 관찰

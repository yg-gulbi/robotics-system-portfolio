# Lab Plan

## 목적

책으로 학습한 컴퓨터구조와 운영체제 개념을 C, Python, Linux 실습으로 확인합니다.

## 실습 목록

| 실습 | 목표 | 도구 |
| --- | --- | --- |
| 프로그램 실행 과정 | 컴파일, 실행, 종료 흐름 확인 | C, Linux |
| 메모리 구조 확인 | stack, heap, data 영역 관찰 | C |
| 프로세스 실습 | `fork`, `exec`, `wait` 이해 | C, Linux |
| 스레드 실습 | pthread 기반 실행 흐름 비교 | C |
| Race Condition | 공유 변수 충돌과 Mutex 확인 | C |
| 가상 메모리 | 페이지 폴트와 메모리 사용량 관찰 | Linux |
| 파일 시스템 | 파일 디스크립터와 입출력 확인 | C, strace |
| 스케줄링 시뮬레이션 | FCFS, SJF, RR 비교 | Python |

## 결과 정리 방식

- 실습 목적
- 실행 코드
- 실행 결과
- 관찰한 점
- 운영체제 개념과의 연결
- 다음 보완점

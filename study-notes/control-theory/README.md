# Control Theory

## 학습 목적

PID, 상태공간, LQR을 단순 공식이 아니라 균형 로봇 제어와 연결해 이해하는 것이 목표입니다.

## 핵심 개념

- 피드백 제어
- PID 제어
- 상태공간 표현
- LQR
- 시스템 안정성

## 정리 내용

PID 제어는 오차를 비례, 적분, 미분 항으로 나누어 제어 입력을 계산합니다. 균형 로봇에서는 기울기와 각속도 데이터를 바탕으로 모터 입력을 조절하는 데 연결됩니다.

상태공간과 LQR은 시스템 상태를 더 구조적으로 다루기 위한 학습 주제입니다. 현재는 개념을 정리하고, 이후 밸런스 로봇의 자세 제어 실험과 연결할 예정입니다.

## 실습 계획

- PID 게인 변화에 따른 응답 비교
- MATLAB 또는 Python 기반 간단한 제어 시뮬레이션
- 밸런스 로봇의 IMU 데이터와 제어 입력 기록
- LQR 적용 가능성 검토

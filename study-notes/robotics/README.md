# Robotics

## 학습 목적

ROS, Gazebo, 센서, SLAM, Navigation을 로봇 프로젝트 확장 방향과 연결해 정리합니다.

## 핵심 개념

- ROS 토픽과 노드
- Gazebo 시뮬레이션
- 센서 데이터 흐름
- SLAM
- Navigation
- 실제 로봇과 시뮬레이션 연결

## 정리 내용

ROS는 로봇 시스템을 여러 노드로 나누고 토픽을 통해 데이터를 주고받는 구조를 제공합니다. Gazebo는 로봇 모델과 센서를 가상 환경에서 실험할 수 있게 해 줍니다.

현재는 기본 구조를 학습 중이며, 이후 밸런스 로봇 프로젝트의 시뮬레이션과 실제 제어 흐름을 연결하는 방향으로 확장할 예정입니다.

## 실습 계획

- 기본 ROS 노드 작성
- 센서 토픽 구독과 발행
- Gazebo 로봇 모델 구성
- 제어 노드와 시뮬레이션 실험
- ROS2 전환 학습

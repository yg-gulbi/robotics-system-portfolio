# Languages

언어는 읽기, 쓰기, 말하기 기준으로 현재 근거가 있는 만큼만 표시합니다. 시험 점수나 자격증이 없으면 없다고 적고, 학습 로그나 문서 작성 근거가 생기면 날짜별로 갱신합니다.

## 평가 기준

| 점수 | 기준 |
| --- | --- |
| 0 | 공개 기록 없음 |
| 1 | 문자, 기초 표현, 짧은 문장 학습 단계 |
| 2 | 짧은 자료를 읽거나 간단한 문장을 작성할 수 있는 단계 |
| 3 | 기술 문서 일부를 읽고 README나 요약 문장에 활용하는 단계 |
| 4 | 독립적으로 자료를 읽고 정리하거나 대화할 수 있는 단계 |
| 5 | 시험, 자격증, 업무 사용 같은 외부 근거가 있는 단계 |

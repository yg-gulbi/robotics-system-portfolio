# Languages

언어 역량은 점수나 자격증을 과장해 쓰기보다, 학습 기록과 기술 학습에 어떻게 연결되는지 중심으로 정리합니다.

## 구성

- [japanese.md](japanese.md): 일본어 학습 기록과 표현 정리
- [english.md](english.md): 기술 문서와 GitHub README 읽기/작성에 필요한 영어 기록

## 작성 기준

- 실제 시험 점수나 자격증이 없으면 점수 표현을 쓰지 않습니다.
- 학습 중인 내용은 학습 중이라고 씁니다.
- 취업 포트폴리오에서는 언어 자체보다 기록 습관, 자료 이해, 커뮤니케이션 가능성을 중심으로 연결합니다.

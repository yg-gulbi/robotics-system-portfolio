# Skill Summary

## Programming

C, C++, Python, MATLAB을 학습하고 있으며, C와 Arduino는 실제 로봇 제어 펌웨어, Python은 ROS node와 실험 자동화, MATLAB은 제어 계산과 학습 코드 복습에 활용하고 있습니다.

## Embedded

Arduino, IMU, ODrive, BLDC Motor, RC receiver를 중심으로 센서 입력, PWM filtering, 모터 전류 명령, 안전 조건을 실제 로봇에서 다뤘습니다.

## Robotics

ROS와 Gazebo를 사용해 simulation, topic flow, SLAM, navigation command path를 정리했습니다. ROS2는 전환 방향과 기본 구조를 학습 중입니다.

## Control

PID, body angle feedback, speed loop, steering correction, safety layer를 밸런싱 로봇의 자세 제어와 연결해 설명합니다.

## System/CS

컴퓨터구조와 운영체제는 책 1회독 후 C/Linux 실습으로 연결할 계획이며, 프로세스, 스레드, 메모리, 시스템 콜을 실습 단위로 정리할 예정입니다.

## AI/RL

강화학습과 딥러닝은 완성 제품이 아니라 Q-learning, DDPG, Input Shaper, MATLAB 실험을 통한 응용 학습 근거로 정리합니다.

## Mechanical/3D

CAD, assembly, STL/STEP export, 3D printing workflow를 밸런싱 로봇의 sensor case, board plate, battery mount, packaging 근거와 연결합니다.

# Skill Summary

## Language

C, C++, Python, MATLAB을 학습하고 있으며, C와 Arduino는 로봇 제어 실험, Python은 실습 자동화와 시뮬레이션 정리에 활용하고 있습니다.

## Embedded

Arduino, IMU, BLDC Motor, ODrive를 중심으로 센서 입력과 모터 제어를 실험하고 있습니다.

## Robotics

ROS와 Gazebo를 학습하며 로봇 모델링, 센서 토픽, 제어 노드 구조를 정리하고 있습니다. ROS2는 전환 방향을 학습 중입니다.

## Control

PID 제어를 밸런스 로봇 자세 제어와 연결해 실험하고 있으며, 상태공간과 LQR은 추가 학습 및 적용 예정입니다.

## System

컴퓨터구조와 운영체제는 책 1회독 후 C/Linux 실습으로 연결하고 있으며, 프로세스, 스레드, 메모리, 시스템 콜을 실습 단위로 정리할 예정입니다.

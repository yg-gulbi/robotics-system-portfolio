# Computer Architecture OS Lab

## 프로젝트 개요

컴퓨터구조와 운영체제 개념을 C, Python, Linux 실습으로 검증하는 학습 프로젝트입니다.

## 문제 정의

컴퓨터구조와 운영체제 개념은 책으로 이해한 것처럼 보여도 실제 프로그램 실행, 메모리 사용, 프로세스/스레드 동작을 직접 확인하지 않으면 체감하기 어렵습니다.

## 목표

- 책으로 학습한 내용을 실제 실습으로 체감
- 프로그램 실행 과정 이해
- 메모리 구조 확인
- 프로세스와 스레드 차이 실습
- Race Condition과 Mutex 확인
- 가상 메모리와 페이지 폴트 체감
- 파일 디스크립터와 시스템 콜 추적
- CPU 스케줄링 알고리즘 시뮬레이션

## 사용 기술

- C
- Python
- Linux
- gcc
- pthread
- strace

## 시스템 구조

```text
Book Concept
  -> Small C/Python Lab
  -> Linux Observation
  -> Result Note
  -> Portfolio Summary
```

## 구현 내용

현재는 실습 항목을 설계하고 있으며, 각 실습은 하나의 작은 코드와 README로 정리할 예정입니다.

## 실행 방법

실습 코드가 추가되면 각 하위 폴더에 실행 명령을 작성합니다.

## 결과

진행 중 프로젝트입니다. 먼저 학습 주제와 실습 계획을 정리하고, 이후 코드와 실행 결과를 단계적으로 추가합니다.

## 배운 점

- 운영체제 개념은 프로그램 실행 흐름과 연결할 때 더 명확해집니다.
- C/Linux 실습은 로봇/임베디드 개발의 기초 체력을 만드는 데 도움이 됩니다.

## 개선할 점

- 실습 코드 추가
- 실행 결과 캡처
- 시스템 콜 추적 기록
- 스케줄링 시뮬레이션 결과 비교

## 포트폴리오 요약

컴퓨터구조와 운영체제의 핵심 개념을 C/Python/Linux 환경에서 실습하고 문서화한 프로젝트입니다.

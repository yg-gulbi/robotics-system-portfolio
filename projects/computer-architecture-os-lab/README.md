# Computer Architecture / OS Lab

## 분류

이 문서는 완성된 프로젝트가 아니라 컴퓨터구조와 운영체제 개념을 C, Python, Linux 실습으로 확인하기 위한 학습/실습 기록입니다.

## 학습 목적

컴퓨터구조와 운영체제 개념은 책으로 이해한 것처럼 보여도 실제 프로그램 실행, 메모리 사용, 프로세스/스레드 동작을 직접 확인하지 않으면 체감하기 어렵습니다.

## 목표

- 책으로 학습한 내용을 작은 실습으로 확인
- 프로그램 실행 과정 이해
- 메모리 구조 확인
- 프로세스와 스레드 차이 실습
- Race Condition과 Mutex 확인
- 가상 메모리와 페이지 폴트 체감
- 파일 디스크립터와 시스템 콜 추적
- CPU 스케줄링 알고리즘 시뮬레이션

## 사용 기술

- C
- Python
- Linux
- gcc
- pthread
- strace

## 실습 흐름

```text
Book Concept
  -> Small C/Python Lab
  -> Linux Observation
  -> Result Note
  -> Portfolio Summary
```

## 현재 상태

현재는 실습 항목을 설계하고 있으며, 각 실습은 하나의 작은 코드와 README로 정리할 예정입니다. 실제 코드와 실행 결과가 충분히 쌓이기 전까지는 프로젝트 목록의 중심 카드로 올리지 않습니다.

# Climbing AI Training System

## 분류

이 문서는 구현 완료 결과물이 아니라 클라이밍 훈련 루틴 추천을 구상하기 위한 아이디어 기록입니다. 현재 포트폴리오의 프로젝트 목록에는 올리지 않고, 기술 밖의 관심과 자동화 아이디어를 보여주는 보조 기록으로 둡니다.

## 아이디어 개요

사용자의 컨디션, 부상 정도, 목표 난이도에 따라 클라이밍 훈련 루틴을 추천하는 시스템을 구상합니다.

## 문제 정의

클라이밍 훈련은 사용자의 현재 컨디션, 부상 위험, 목표 난이도에 따라 달라져야 합니다. 초보자는 어떤 훈련을 선택해야 할지 판단하기 어렵고, 무리한 훈련은 부상으로 이어질 수 있습니다.

## 구상 목표

- 사용자 컨디션 입력 구조 설계
- 부상 정도와 목표에 따른 훈련 루틴 추천 규칙 정리
- Notion, Make, AI 모델을 활용한 간단한 자동화 가능성 검토
- 운동 기록과 추천 결과를 누적하는 구조 구상

## 사용할 수 있는 도구

- Notion
- Make
- AI 모델
- Python 또는 웹 자동화 도구

## 구상 흐름

```text
User Condition
  -> Goal / Injury Check
  -> Routine Recommendation
  -> Notion Record
  -> Feedback
```

## 현재 상태

현재는 문제 정의와 입력 항목을 구상하는 단계입니다. 구현 전 단계이므로 완성된 서비스처럼 표현하지 않습니다.

## 다음 보완점

- 입력 항목 구체화
- 추천 규칙 초안 작성
- Notion 데이터베이스 구조 설계
- 간단한 MVP 검증 방식 정리

## 포트폴리오 요약

컨디션과 목표에 따라 클라이밍 훈련 루틴을 추천하는 자동화 아이디어 기록입니다.

# Climbing AI Training System

## 프로젝트 개요

사용자의 컨디션, 부상 정도, 목표에 따라 클라이밍 훈련 루틴을 추천하는 시스템 아이디어입니다.

## 문제 정의

클라이밍 훈련은 사용자의 현재 컨디션, 부상 위험, 목표 난이도에 따라 달라져야 합니다. 하지만 초보자는 어떤 훈련을 선택해야 할지 판단하기 어렵고, 무리한 훈련은 부상으로 이어질 수 있습니다.

## 목표

- 사용자 컨디션 입력 구조 설계
- 부상 정도와 목표에 따른 훈련 루틴 추천
- Notion, Make, AI 모델을 활용한 간단한 자동화 가능성 검토
- 향후 운동 기록과 추천 결과를 누적하는 구조 구상

## 사용 기술

- Notion
- Make
- AI 모델
- Python 또는 웹 자동화 도구

## 시스템 구조

```text
User Condition
  -> Goal / Injury Check
  -> Routine Recommendation
  -> Notion Record
  -> Feedback
```

## 구현 내용

현재는 아이디어 구체화 단계입니다. 로봇/임베디드 중심 포트폴리오 안에서는 주력 프로젝트가 아니라 AI/자동화 도구 활용 가능성을 보여주는 보조 프로젝트로 둡니다.

## 실행 방법

구현 전 단계입니다. 입력 항목과 추천 규칙을 먼저 문서화한 뒤 간단한 MVP로 확장합니다.

## 결과

아이디어 단계이며, 완성된 서비스처럼 표현하지 않습니다.

## 배운 점

- 기술 포트폴리오에서는 아이디어도 문제 정의와 검증 계획이 함께 있어야 설득력이 있습니다.
- 주력 분야와 거리가 있는 프로젝트는 보조 프로젝트로 위치를 명확히 두는 것이 좋습니다.

## 개선할 점

- 입력 항목 구체화
- 추천 규칙 초안 작성
- Notion 데이터베이스 구조 설계
- 간단한 MVP 검증 방식 정리

## 포트폴리오 요약

컨디션과 목표에 따라 클라이밍 훈련 루틴을 추천하는 AI/자동화 기반 아이디어 프로젝트입니다.

# Profile

## 자기소개

로봇 SW와 임베디드 시스템 개발자를 목표로 공부하고 있습니다. 센서 데이터, 제어, 하드웨어 동작, 운영체제 기초를 따로 보지 않고 실제 프로젝트 안에서 연결해 이해하는 방향을 지향합니다.

아직 모든 프로젝트가 완성 단계는 아니지만, 진행 중인 내용을 숨기기보다 문제 정의, 실험 과정, 배운 점, 다음 보완점을 문서로 남기고 있습니다.

## 관심 분야

- 로봇 제어와 센서 데이터 처리
- Arduino, IMU, 모터 제어 기반 임베디드 시스템
- ROS/Gazebo 기반 로봇 시뮬레이션
- C/Linux 기반 운영체제 실습
- 시스템 프로그래밍과 하드웨어 가까운 개발

## 목표 직무

- 로봇 소프트웨어 개발
- 임베디드 시스템 개발
- 시스템 프로그래밍 기초가 필요한 로봇/제어 분야 개발

## 학습 방향

책이나 강의로 익힌 개념을 단순 요약에서 끝내지 않고, 작은 실습과 프로젝트로 검증하는 방향으로 정리합니다.

## 포트폴리오 전체 요약

이 포트폴리오는 로봇 SW, 임베디드, 운영체제 기초를 공부하며 프로젝트로 연결해 가는 과정을 보여주는 문서형 저장소입니다.

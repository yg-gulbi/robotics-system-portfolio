# Profile

## 자기소개

로봇 SW와 임베디드 시스템 개발자를 목표로 공부하고 있습니다. 센서 데이터, 제어, 하드웨어 동작, 운영체제 기초를 따로 보지 않고 실제 로봇과 학습 기록 사이의 연결로 이해하는 방향을 지향합니다.

공부 중인 내용을 프로젝트처럼 포장하기보다, 실제 결과물이 있는 것은 프로젝트로 두고 나머지는 지식 지도와 실습 기록으로 분리합니다. 진행 중인 내용을 숨기지 않고 문제 정의, 실험 과정, 배운 점, 다음 보완점을 문서로 남깁니다.

## 관심 분야

- 로봇 제어와 센서 데이터 처리
- Arduino, IMU, 모터 제어 기반 임베디드 시스템
- ROS/Gazebo 기반 로봇 시뮬레이션
- C/Linux 기반 운영체제 실습
- 시스템 프로그래밍과 하드웨어 가까운 개발

## 목표 직무

- 로봇 소프트웨어 개발
- 임베디드 시스템 개발
- 시스템 프로그래밍 기초가 필요한 로봇/제어 분야 개발

## 학습 방향

책이나 강의로 익힌 개념을 단순 요약에서 끝내지 않고, 작은 실습과 실제 프로젝트 근거로 나누어 검증하는 방향으로 정리합니다.

## 기술 밖의 관심

언어를 배우고, 클라이밍을 하며 몸의 감각을 살피고, 생각을 문장으로 정리하는 일을 좋아합니다. 이 관심들은 로봇 개발 자체는 아니지만 문제를 관찰하고 기록하는 방식에 영향을 줍니다.

## 포트폴리오 전체 요약

이 포트폴리오는 실제 로봇 프로젝트와 학습 지도를 구분해 정리하는 문서형 저장소입니다.

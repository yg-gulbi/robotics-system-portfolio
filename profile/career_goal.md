# Career Goal

## 목표

로봇 SW와 임베디드 시스템 개발자로 성장하기 위해 제어, 센서, ROS, 운영체제 기초를 꾸준히 실습하고 문서화합니다.

## 단기 목표

- self-balancing-robot 프로젝트의 센서 입력, 제어, 모터 구동 흐름 정리
- 컴퓨터구조/운영체제 개념을 C/Linux 실습으로 연결
- ROS/Gazebo 기반 시뮬레이션 실습 정리
- 기술 스택 수준을 근거 중심으로 갱신

## 중기 목표

- 실제 하드웨어 제어와 시뮬레이션 결과를 비교
- ROS2, SLAM, Navigation 학습 자료 확장
- 시스템 콜, 프로세스, 스레드, 동기화 실습을 작은 프로젝트로 정리

## 문서화 기준

- 완성된 것처럼 과장하지 않습니다.
- 진행 중인 내용은 진행 중이라고 씁니다.
- 사용 기술, 문제 정의, 실험 결과, 다음 보완점을 함께 남깁니다.
